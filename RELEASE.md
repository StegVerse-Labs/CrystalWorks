# Release Process (CrystalWorks)

## Core Compatibility
Pin adopted DiamondOps-Core version here:
- Core compatibility: v0.1.x

## Versioning
- PATCH: doc fixes
- MINOR: new program templates, measurement frameworks
- MAJOR: governance boundary changes

## Release steps
1. Ensure inclusion/exclusion criteria remain intact
2. Add/adjust templates and measurement plans
3. Tag GitHub Release `vX.Y.Z`
